need-a-doc
============

Run locally
------------

### Dependencies

[nodeJS](https://nodejs.org/)

[yarn](https://yarnpkg.com/)


### Install

```bash
yarn
```

### Run

```bash
yarn run app
```

Project decisions
-----------------

- Since it's a simple project, front and backend code are in the same repo to improve coding speed
- PugJS was the chosen frontend framework for simplicity
